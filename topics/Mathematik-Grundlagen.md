# Mathematik-Grundlagen

## Exponenten Rechenregeln

$a^2$

$a^{\frac{1}{2}} = \sqrt[2]{a}$

$a^{\frac{1}{n}} = \sqrt[n]{a}$

$a^{\frac{3}{12}} = \sqrt[12]{a^3}$

$a^{-2} = \frac{1}{a^2}$

$a^{-n} = \frac{1}{a^n}$

$2^{-\frac{1}{12}} = \frac{1}{\sqrt[12]{2}}$

$2^{-\frac{12}{12}} = \frac{1}{\sqrt[12]{2^{12}}} = \frac{1}{2}$

$a^0 = 1$

$a^1 = a$
